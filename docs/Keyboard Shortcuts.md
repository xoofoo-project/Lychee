### Everywhere
| Key | Action |
|:-----------|:------------|
| `enter` | Confirm Dialog |
| `u` | Upload photo |
| `esc` | Close/Back |  
| `cmd`+`up` | Close/Back | 

### Album
| Key | Action |
|:-----------|:------------|
| `i` | Show information |
| `cmd`+`backspace` | Delete album |

### Photo
| Key | Action |
|:-----------|:------------|
| `s` | Star photo |  
| `i` | Show information |  
| `cmd`+`backspace` | Delete photo |
| `left` | Previous photo |
| `right` | Next photo |
